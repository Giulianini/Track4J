# Track4J

## A simple framework for body tracking in Java.

**Supported sensors:** 

* **Windows:** 
	- _Kinect v1_
	- _Kinect v2_

**Sooner or Later:**

* **Windows:** 
	- _Lipmotion_
	
* **Linux:**
	- _Kinect v1_
	- _Kinect v2_
